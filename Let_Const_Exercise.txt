Quiz

- What is the difference between var and let?
Var has a scope of function level while let has a scope of block level.
Var can be redeclared, but let can not.

- What is the difference between var and const?
Var has a scope of function level while const has a scope of block level.
Var can be redeclared and reassigned, but const can not.

- What is the difference between let and const?
let can be reassigned, but const can not.

- What is hoisting?
When the code is compiled, the variable are lifted to the top of the scope they are declared in.
'var' allows you to access the variable.
